About Saidos Workflow

Saidos Workflow is a professional social media automation platform designed for content creators, marketing agencies, and businesses. We provide a reliable and GDPR-compliant bridge to automate content publishing across all major social media platforms via their official APIs.

Our primary integrations are the Meta Graph API (Facebook & Instagram) and the YouTube Data API v3, extended by TikTok, LinkedIn, Pinterest, and X (Twitter) — giving you a true all-in-one content posting solution.

1. Professional Use Case

Saidos Workflow provides an automated content management solution for professionals. Our platform strictly follows the API guidelines of all connected platforms — including Meta Platform Terms, YouTube API Services Terms of Service, and TikTok's Developer Terms — to ensure a safe, transparent, and compliant environment for all users.

2. Supported Platforms & APIs

We currently support the following platforms and their official APIs for automated content posting:

Meta Graph API Facebook Pages & Instagram Business — content posting, scheduling, insights
YouTube Data API v3 Video uploads, channel management, playlist handling
TikTok API Video content publishing and account management
LinkedIn API Posts to profiles and company pages
Pinterest API Pin creation and board management
X (Twitter) API Tweet posting and content scheduling

3. Meta Graph API – Key Integration

Our core integration with the Meta Graph API enables fully automated content publishing to Facebook Pages and Instagram Business accounts. We request only the minimum permissions required:

  • pages_manage_posts – Publish posts to Facebook Pages
  • instagram_content_publish – Publish media to Instagram Business accounts
  • pages_read_engagement – Read basic post engagement data

We do not access personal Facebook profiles, friend lists, private messages, or any data beyond the permissions listed above. All access tokens are stored encrypted and are never shared with third parties.

4. YouTube Data API v3 – Key Integration

Our integration with the YouTube Data API v3 (provided by Google LLC) enables automated video uploads and channel management. Users can revoke access at any time via Google Security Settings. By using this feature, users agree to the YouTube Terms of Service and Google Privacy Policy.
