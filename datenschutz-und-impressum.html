Datenschutzerklärung

Wir freuen uns sehr über Ihr Interesse an unserem Unternehmen. Datenschutz hat einen besonders hohen Stellenwert für die Geschäftsleitung der Saidos Workflow. Eine Nutzung der Internetseiten der Saidos Workflow ist grundsätzlich ohne jede Angabe personenbezogener Daten möglich. Sofern eine betroffene Person besondere Services unseres Unternehmens über unsere Internetseite in Anspruch nehmen möchte, könnte jedoch eine Verarbeitung personenbezogener Daten erforderlich werden.

Unser Dienst ermöglicht die automatisierte Veröffentlichung von Inhalten auf Social-Media-Plattformen über deren offizielle APIs – mit Schwerpunkt auf der Meta Graph API (Facebook & Instagram) und der YouTube Data API v3, ergänzt durch TikTok, LinkedIn, Pinterest und X (Twitter).

Die Verarbeitung personenbezogener Daten erfolgt stets im Einklang mit der Datenschutz-Grundverordnung (DSGVO) und den für die Saidos Workflow geltenden landesspezifischen Datenschutzbestimmungen.

1. Begriffsbestimmungen

Die Datenschutzerklärung der Saidos Workflow beruht auf den Begrifflichkeiten der Datenschutz-Grundverordnung (DS-GVO):

  • a) Personenbezogene Daten

    Alle Informationen, die sich auf eine identifizierte oder identifizierbare natürliche Person beziehen, einschließlich Name, Kennnummer, Standortdaten oder Online-Kennung.

  • b) Betroffene Person

    Jede identifizierte oder identifizierbare natürliche Person, deren personenbezogene Daten vom Verantwortlichen verarbeitet werden.

  • c) Verarbeitung

    Jeder Vorgang im Zusammenhang mit personenbezogenen Daten – Erheben, Speichern, Verändern, Übermitteln, Löschen usw.

  • d) Einschränkung der Verarbeitung

    Markierung gespeicherter personenbezogener Daten zur Einschränkung ihrer künftigen Verarbeitung.

  • e) Profiling

    Automatisierte Verarbeitung personenbezogener Daten zur Bewertung bestimmter persönlicher Aspekte einer natürlichen Person.

  • f) Pseudonymisierung

    Verarbeitung personenbezogener Daten, sodass diese ohne Hinzuziehung zusätzlicher Informationen nicht mehr einer spezifischen Person zugeordnet werden können.

  • g) Verantwortlicher

    Die natürliche oder juristische Person, die über Zwecke und Mittel der Verarbeitung von personenbezogenen Daten entscheidet.

  • h) Auftragsverarbeiter

    Eine natürliche oder juristische Person, die personenbezogene Daten im Auftrag des Verantwortlichen verarbeitet.

  • i) Empfänger

    Eine natürliche oder juristische Person, der personenbezogene Daten offengelegt werden.

  • j) Dritter

    Jede Stelle außer der betroffenen Person, dem Verantwortlichen und dem Auftragsverarbeiter.

  • k) Einwilligung

    Jede freiwillig und unmissverständlich abgegebene Willensbekundung der betroffenen Person zur Verarbeitung ihrer personenbezogenen Daten.

3. Nutzung der Meta Graph API (Facebook & Instagram)

Unser Dienst nutzt die Meta Graph API, um im Auftrag des Nutzers automatisch Inhalte auf Facebook-Seiten und Instagram-Business-Konten zu veröffentlichen. Wir fordern ausschließlich die für diesen Zweck notwendigen Berechtigungen an:

  • pages_manage_posts – Veröffentlichen von Beiträgen auf Facebook-Seiten
  • instagram_content_publish – Veröffentlichen von Medien auf Instagram
  • pages_read_engagement – Lesen grundlegender Interaktionsdaten
Wir haben keinen Zugriff auf persönliche Facebook-Profile, Freundeslisten oder private Nachrichten. Zugriffstoken werden verschlüsselt gespeichert und niemals an Dritte weitergegeben.

Mit der Verbindung deines Meta-Kontos stimmst du zusätzlich den Meta-Nutzungsbedingungen und den Meta Platform Terms zu.

4. Nutzung der YouTube Data API v3

Unser Dienst integriert die YouTube Data API v3 (bereitgestellt von Google LLC) für automatisierte Video-Uploads und Kanalverwaltung. Mit der Nutzung dieser Funktion stimmst du den YouTube-Nutzungsbedingungen und der Google-Datenschutzerklärung zu. Du kannst den Zugriff jederzeit über die Google-Sicherheitseinstellungen widerrufen.

5. Nutzung weiterer Social-Media-APIs

Unser Dienst kann zusätzlich folgende Plattform-APIs für Content-Posting und Automatisierung nutzen:

  • TikTok API for Developers – Automatisiertes Video-Publishing. Es gelten die TikTok-Nutzungsbedingungen.
  • LinkedIn API – Veröffentlichung auf LinkedIn-Profilen und Unternehmensseiten. Es gilt das LinkedIn-Nutzervertrag.
  • Pinterest API – Pin-Erstellung und Board-Verwaltung. Es gelten die Pinterest-Nutzungsbedingungen.
  • X (Twitter) API – Tweet-Posting und Content-Scheduling. Es gelten die X-Nutzungsbedingungen.

6. Welche Daten wir über APIs verarbeiten

Wenn du ein Social-Media-Konto mit unserem Dienst verbindest, verarbeiten wir folgende Datenkategorien:

  • Plattform-Nutzer-ID und Benutzername
  • Kontoname und Profilbild-URL
  • Zugriffstoken und Aktualisierungstoken (verschlüsselt gespeichert)
  • Seiten- oder Kanal-IDs, die mit deinem Konto verknüpft sind
  • Von dir übermittelte Inhalte (Texte, Bilder, Videos) zur Veröffentlichung
  • Grundlegende Interaktionsdaten – nur wenn du die entsprechende Berechtigung erteilt hast
